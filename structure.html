HTML Fundamentals

The <!DOCTYPE html> tag is required for every html page so that the web page is viewed in the same way, even in different browsers.

The <html lang='en'> tag is used to make sure that the page will be displaying html. The lang='en' part is to let the page know that the website is in english.


The <head> tag is used to include the title of your website and other items that aren't physically shown on the website itself.
