HTML Tags

The tags in the snippet HTML above are: p and a.

References:

  • W3Schools P tag Reference!

The a tag is a child of the p tag. Whatever is put in the p tag is displayed on the site.

The opening p is to start the paragraph on the page. The closing p with the forward slash ends the paragraph on the page.

The opening a is to start the process of adding a hyperlink on the page. The closing a with the forward slash ends the hyperlink process on the page.

HTML Attributes

The class attribute links to a class name in a stylesheet. The href attribute makes sure the URL of the page is linked to the site. The id attribute clarafies a the id for an element. The target attribute clarafies where to open the link or form.
