Styling a Page

This is a copy of the basic-tags.html except I added some CSS.

Below is an example of a h2 tag. The h2 tag is used to display another header that is slightly smaller than the h1 tag header.

Daily Schedule

Below is an example of the paragraph tag. The paragraph tag is used to display text on the website.

Monday-Go to Biology class.

Below is an example of the a tag. The a tag is used to display a hyperlinked text that will take the user to a different webpage.

Monday

Below is an example of the blockquote tag. The blockquote tag is used to cite a source for a quote that is being used in your website. Within the blockquote tag I have also included a i tag. The i tag is used to define a a piece of text in a different mood.

Agnes Scott offers the following majors, minors and academic programs. Unless otherwise noted, majors result in a Bachelor of Arts degree.

Below is an example of the pre tag. The pre tag is used to display a preformatted text on the website. Within the pre tag I have also included an em tag. The em tag is used to emphasize a piece of text.

      Buy more pencils after work.

Below is an example of the strong tag. The strong tag is used to define important text. The b tag can also do the same function as the strong tag. Within this example I have also used the code tag. The code tag is used to define a piece of text that has been coded.

Go buy to purchase something groceries!

Below is an example of the ul tag. The ul tag is used to display an unordered list on the website. Within this ul I have also included a img tag. The img tag helps define an image on the website.

  • Apples
  • Oranges
  • Spinach
  • Grilled Chicken

Below is an example of the ol tag. The ol tag is used to display an ordered list on the website. Within this ol I have inserted another hyperlink with the a tag.

  1. Mangos
  2. Watermelon
  3. Turkey Bacon
  4. Bread

Below is another example of a unordered list but I have now nested another unordered list within another one.

  • Coffee
  • Tea
    • Peach tea
    • Green tea
  • Milk

Below is an example of the table tag. The table tag is used to create a text table on the website. The th tag is used for the table headinngs. The tr tag is used to define a row in the table. The td tag is used to define a cell in the table.

Weekday Homework
Monday Write new vocab and definitions.
Tuesday Work on problem set.
Wednesday Write my midterm essay.
Thursday Read chapter 6 for next class.
Friday Go walk 2 miles on the track.
